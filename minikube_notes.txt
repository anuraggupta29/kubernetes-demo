1. Install Docker Desktop + Linux WSL for windows
2. Install Minikube
3. Install kubectl
4. Set environment variable for Minikube and kubectl (the folder where minikube.exe and kubectl.exe is present)
setx PATH "$env:PATH;C:\tools\kubectl;C:\tools\minikube"

5. Run minikube from CMD
minikube start --driver=docker

6. Success response:
